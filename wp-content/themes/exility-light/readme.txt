= Exility-light =

Exility-light is an elegant responsive WordPress  theme.  Features:  built-in slider, 2 Columns, 2 Sidebars, custom menu,  post thumbnails, page navigation, widget ready.

Slider shows sticky posts.
Mark your answer several posts as sticky.
Under side menu of post/ "Publish" / "Visibility". Click "Edit"/ Check Box "Stick this post to the front page"
